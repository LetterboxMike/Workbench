# Notion-Level Minimal Direction

Use one direction: restrained, editorial, calm utility.

## Core Style

- Keep visual noise low.
- Let spacing, typography, and alignment do most of the work.
- Prefer borders and contrast shifts over strong shadows.
- Keep accent color usage rare and functional.
- Use compact but breathable density for B2B throughput.

## Non-Negotiable Color Rule

- Never use true black or true white.
- Ban these exact values: `#000`, `#000000`, `rgb(0,0,0)`, `hsl(0 0% 0%)`, `#fff`, `#ffffff`, `rgb(255,255,255)`, `hsl(0 0% 100%)`.
- Use near-black and near-white grays instead.

## Light Mode Gray Ramp (Recommended)

- `--bg`: `#f6f6f4`
- `--surface`: `#efefec`
- `--surface-elevated`: `#e7e7e3`
- `--border`: `#d6d6d1`
- `--text-primary`: `#262624`
- `--text-secondary`: `#4d4d49`
- `--text-muted`: `#6a6a65`
- `--focus-ring`: `#8c8c86`

## Dark Mode Gray Ramp (Recommended)

- `--bg`: `#171715`
- `--surface`: `#1e1e1c`
- `--surface-elevated`: `#262624`
- `--border`: `#353532`
- `--text-primary`: `#e5e5e1`
- `--text-secondary`: `#bebeb8`
- `--text-muted`: `#9b9b94`
- `--focus-ring`: `#a4a49e`

## Semantic Colors (Muted)

- Keep semantics desaturated and close to neutral.
- Ensure semantic values also avoid pure black and pure white.
- Keep semantic contrast sufficient in both modes.

## Type and Rhythm

- Prefer readable sans-serif body type with modest personality.
- Keep display styles subtle and functional.
- Use a tight spacing scale with consistent increments.
- Maintain predictable line heights for tables and forms.

## Motion

- Use short transitions only for state change clarity.
- Avoid theatrical entrance animations.
- Keep motion duration consistent and minimal.
